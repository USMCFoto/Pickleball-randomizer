This is an app for people to use when playing pickleball doubles. 
It is meant to generate a nearly random sets of doubles for whatever number of games, players and courts people want to use.
Nearly random that no players should be paired together for a 2nd+ time until all other pairings have been exhausted.
Byes for skipping games should be random except that nobody will be placed on a 2nd+ bye until all players have had an individual bye.
